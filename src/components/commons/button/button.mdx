# Button

The `Button` component provides a flexible and customizable button for various use cases in your UI. Built by extending `HTMLButtonElement`, it inherits all standard `HTML` button attributes, so you can apply native `HTML` button props like `onClick`, `disabled`, `type`, `aria-\*` attributes, and more, making it highly adaptable to different requirements. Additionally, the `Button` component offers specific props such as `label`, `loading` state, `rounded` edges, `icons`, and multiple `size` and `variant` options. Designed with accessibility in mind, it supports keyboard navigation and screen readers, ensuring it integrates seamlessly into any project.

### Default Usage

The `Button` component is simple to use. Just add the `Button` component and set properties like label for text. This makes it ideal for common use cases such as form submission or navigating between pages.

### Label

The `label` prop specifies the text displayed on the button. This prop is required to give context to the button’s action.

<div class="custom-canvas">
  <Story of={ButtonStories.ButtonLabel} />
</div>

<Source
  dark
  code={`

Use the `loading` prop to indicate that an action is in progress. When set to true, the button shows a loading indicator and becomes `disabled`.

### Variant

The `variant` prop allows customization of the button’s style. Buttons can have different variant colors by setting the `variant` prop. The default variant is `informative`.

<Button label="Success Button" variant="success" />

The `align` prop controls the alignment of the button's content (label and icons). Use it to set the alignment to `left`, `center`, or `right` to better suit your design requirements.

### Button With Icon

The `icon` prop allows you to add an `icon` on either the left or right side of the label for added context. This is useful for actions that benefit from a visual cue.

  <Story of={ButtonStories.ButtonIconRight} />
</div>

### Children

The `children` prop allows you to fully customize the button's content. Instead of a `label`, you can pass JSX elements such as text wrapped in a `<span>`, icons, or even a combination of both. This gives you full control over the button's design and content.

<div class="custom-canvas">
  <Story of={ButtonStories.ButtonChildren} />
</div>

<Source
  dark

  <span>Click Me</span>
</Button>
`}
/>

### Properties

- **`<Button>`** properties:

<Markdown>
  {`
| prop            | type                            | default     | description 
|-----------------|---------------------------------|-------------|-------------
| className       | string                          | -           | Custom class name for styling the button.
| classNameLabel  | string                          | -           | Custom class name for styling the button's label.
| label           | string                          | -           | The text displayed inside the button as its label.
| loading         | boolean                         | false       | Shows a loading state inside the button (e.g., spinner or indicator).
| outline         | boolean                         | false       | Applies an outline style to the button, without background fill.
| rounded         | boolean                         | false       | Adds rounded corners to the button.
| disabled        | boolean                         | false       | Disables the button, preventing interaction.
| fullWidth       | boolean                         | false       | Makes the button span the full width of its container.
| variant         | TypeVariant                     | -           | Defines the style variant of the button (e.g., primary, secondary).
| size            | ButtonSizes                     | -           | Sets the size of the button (e.g., small, medium, large).
| align           | TypeAligns                      | -           | Controls the alignment of the button's label and icon.
| iconLeft        | React.ReactNode                 | -           | Adds an icon to the left side of the button's label.
| iconRight       | React.ReactNode                 | -           | Adds an icon to the right side of the button's label.
| children*       | ReactNode                       | false       | The content inside the button, usually the label text or elements.
| onClick         | () => void                      | -           | A callback function triggered when the button is clicked.
`}
</Markdown>
