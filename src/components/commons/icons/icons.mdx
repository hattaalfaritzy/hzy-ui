# Icons

The `Icons` component is a versatile solution for rendering SVG-based icons in your UI. It allows for customization of size, color, and additional styles via class names or props. The component supports a predefined set of icons, ensuring consistent and efficient usage.

<br />

## Import

To use the `Icons` component, import it from `@hattaalfaritzy/hzy-ui` as shown below.

<Source
  dark
  code={`
import { Icons } from "@hattaalfaritzy/hzy-ui";
`}
/>

## Usage

The `Icons` component provides a flexible way to include SVG icons in your application. You can specify the desired icon by name and customize its size, color, or additional styling.

<br />

### Default Usage

Here is an example of how to render an icon using the `Icons` component. Pass the desired icon name via the `name` prop.

<div class="custom-canvas full">
  <Story of={IconStories.IconDefault} />
</div>

<Source dark code={`
import { Icons } from "@/hattaalfaritzy/hzy-ui";

{" "}

<Icons name="activity" />
`} />

### Customization

You can customize the appearance of an icon by applying additional class names or props, such as size and color. <br />

<span className="important-message">
  *The activity icon can be styled with TailwindCSS stroke classes.*
</span>

<div class="custom-canvas full">
  <Story of={IconStories.IconCustomized} />

<Source

  code={`

<Icons name="activity" className="w-8 h-8 stroke-blue-500" />
`}
/>

### Properties

- **`<Icons>`** properties:

<Markdown>
  {` 
| prop            | type                            | default     | description 
|-----------------|---------------------------------|-------------|-------------
| className       | string                          | -           | Custom class name for styling the SVG icon.
| name*           | IconName                        | -           | The name of the icon to render. Must be one of the predefined keys in the \`icons\` object.
| ...props        | ISvgProps                       | -           | Additional props passed down to the SVG component, such as \`width\`, \`height\`, or \`fill\`.
`}
</Markdown>

To see all properties available for `Icons`, including those inherited from `SVGAttributes<SVGElement>`.<br />
Check the [SVGElement documentation](https://developer.mozilla.org/en-US/docs/Web/SVG/Element) for additional attributes.
