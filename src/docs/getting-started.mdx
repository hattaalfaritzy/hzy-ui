# Getting Started

Hzy UI is a library of ready-to-use, accessible, reusable, and composable React UI components built with the [Tailwind CSS](https://tailwindcss.com/) framework. It aims to simplify the development of modern web applications by providing a set of high-quality, customizable components that adhere to the latest web standards.

## Requirements

Made sure to have the correct versions installed in your project.

- [React](https://react.dev) v18 is required

## Installation

To use Hzy UI in your project, you need to install the package, with npm or yarn, run one of the following commands in your terminal:

```sh
# with Yarn
$ yarn add @hattaalfaritzy/hzy-ui

# with npm
$ npm i @hattaalfaritzy/hzy-ui
```

## Usage

To start using the components, on your application entry point, import the CSS file.

```tsx
import "@hattaalfaritzy/hzy-ui/dist/index.css";
```

Start using the components in your application

```tsx
// Do this at the root of your application
import { Button } from "@hattaalfaritzy/hzy-ui";

const App = () => {
  const handleClick = () => alert("Button Clicked!");

  return (
    <div className="flex justify-center items-center h-screen bg-gray-100">
      <Button
        align="center"
        className="w-full"
        label="Click Me"
        size="md"
        variant="informative"
        onClick={handleClick}
      />
    </div>
  );
};

export default App;
```
